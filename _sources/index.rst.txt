======
Reptar
======

**Rep**\ roducible, **t**\ ailorable **ar**\ chive (i.e., reptar) provides an easy way to store (and hopefully release openly) and analyze computational chemistry data.

Installation
============

At the moment, the only way to install reptar is directly from the `GitHub repository <https://github.com/aalexmmaldonado/reptar>`_.

.. code-block:: bash

    git clone https://github.com/aalexmmaldonado/reptar
    cd reptar
    pip install .
